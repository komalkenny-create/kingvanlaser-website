## 🚀 步骤 2：提交 Sitemap (立即执行)
1.  **打开 Google Search Console**：
    👉 [点击这里打开 GSC](https://search.google.com/search-console)
2.  **选择属性**：点击 `kvlasermarking.com`
3.  **提交 Sitemap**：
    - 点击左侧菜单 **"Sitemap"**
    - 在输入框中输入：`sitemap.xml`
    - 点击 **"提交" (Submit)**
4.  **等待处理**：Google 通常会在几分钟内显示 "Success"。

## 📊 步骤 3：监控收录 (3-7 天后检查)
**方法 A：使用浏览器命令 (最快)**
在浏览器地址栏输入：
```
site:kvlasermarking.com
```
✅ **成功标志**：能看到首页、产品页、About 等页面列表。
❌ **失败标志**：显示 "Your search did not match any documents" (需检查 robots.txt 或重新提交)。

**方法 B：使用 GSC 报告**
1.  进入 GSC -> **"索引"** -> **"网页"**
2.  查看 **"已发现的网页 - 未索引"** 数量。
3.  如果数量为 0 或很少，说明收录正常。

## 📈 步骤 4：查看排名 (持续监控)
**工具推荐**：
1.  **Google Search Console (最准确)**：
    - 进入 GSC -> **"效果"** (Performance)
    - 查看 **"查询"** 标签页，搜索你的关键词（如 `laser marking machine`）。
    - 关注 **"展示次数" (Impressions)** 和 **"点击率" (CTR)**。

2.  **第三方免费工具**：
    - [Ahrefs Free Rank Checker](https://ahrefs.com/rank-checker) (输入域名查看预估排名)
    - [SEMrush Position Tracking](https://www.semrush.com/) (免费版有限制)

**📅 建议检查时间**：
- **第 3 天**：检查是否被收录。
- **第 7 天**：检查核心词是否有展示。
- **第 30 天**：检查长尾词排名是否上升。

## 📞 遇到问题？
- **文件上传后不生效？** 清除浏览器缓存或按 `Ctrl + F5` 强制刷新。
- **Sitemap 提交报错？** 检查 `sitemap.xml` 中的 URL 是否与服务器实际 URL 一致（http vs https）。
- **收录慢？** 在 GSC 中使用 **"网址检查" (URL Inspection)** 工具，输入首页 URL，点击 **"请求编入索引" (Request Indexing)** 加速收录。
